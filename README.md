# PalaviTech Shield

**PalaviTech Shield** is a lightweight, modular command-line security assessment tool designed for Windows and Active Directory environments. It provides enumeration and security insights to support penetration testing, red teaming, blue team defense, and system hygiene assessments.  
Shield is being built incrementally, with new capabilities added over time.
